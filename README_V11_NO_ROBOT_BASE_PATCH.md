# Nong Nam v11 No-Robot Base Patch

แพตช์นี้ทำจากไฟล์ฐาน `nongnam-v8.zip` ที่อัปโหลดล่าสุด ไม่ใช่ patch ลอย ๆ

## ไฟล์ที่ต้องลากทับ/เพิ่ม

```txt
app/page.tsx
app/api/chat/route.ts
lib/nongnamHumanCore.ts
```

## แก้อะไร

1. ตัด localReply เก่าที่มีคำแนว:
   - ฟังอยู่
   - พูดมาเลย
   - เล่าให้ฟังได้เลย

2. เพิ่ม `lib/nongnamHumanCore.ts`
   - อ่านเจตนาข้อความก่อน
   - ถ้าเป็นเคสเสี่ยง เช่น ถามว่าแกล้งอะไร / ถามชีวิตน้องน้ำ / ขอคำแนะนำ / โดนด่า / ตำหนิว่าตอบผิด จะตอบจาก local human reply ทันที ไม่ปล่อยให้ AI ตอบ generic

3. route.ts ใหม่
   - ถ้าเจอ intent สำคัญ จะตอบ `v11-human-local`
   - ถ้าต้องใช้ OpenAI จะมี guard กันคำตอบเหมือน AI
   - ถ้าคำตอบมีคำต้องห้าม จะทิ้งแล้วใช้ fallback ที่ตรง intent

4. page.tsx ใหม่
   - เพิ่ม `clientNonce` เพื่อให้การสุ่มไม่ล็อกซ้ำ
   - เปลี่ยน localReply เก่าให้เป็นภาษาคนมากขึ้น
