# Nong Nam AI-First Human V6

สิ่งที่แก้จากเวอร์ชันเดิม:

1. ตัดระบบตอบสำเร็จรูปออกจากเส้นทางหลัก
- เปิด `apiConsent` เป็นค่าเริ่มต้น
- บังคับ `apiMode` ไม่ให้กลับไป local
- route `/api/chat` ใช้ OpenAI เป็นหลัก
- fallback เหลือแค่ข้อความสั้นกรณี API/key ล้มเหลวเท่านั้น

2. แก้คำทักทายเปิดฉาก
- หลังตั้งค่าเสร็จจะพูดสั้น ๆ เช่น `มาแล้วเหรอค่ะ 💗`
- ไม่ถามชื่อ/วันเกิดรัว ๆ อีก

3. แก้ไมค์
- เปลี่ยนจากกดค้างเป็นแตะครั้งแรกเริ่มฟัง / แตะอีกครั้งหยุดส่ง
- ใช้ Web Speech API แล้วเก็บทั้ง interim/final transcript
- ถ้าไม่มีข้อความ จะขึ้น error ชัดเจน ไม่ fail เงียบ

4. แก้ memory ส่งเข้า AI
- ถ้าผู้ใช้บอกชื่อ เช่น `ฉันชื่อแมน` จะจำเป็น `แมน`
- ส่ง memory ที่อัปเดตแล้วเข้า API ไม่ใช้ค่าค้างเก่า

5. ปรับ persona
- ตอบตรงคำถามก่อน
- ตอบสั้น 1–3 ประโยค
- ถามกลับแบบมีบริบท ไม่ใช่พูด `น้องน้ำฟังอยู่` ซ้ำ ๆ
- ถามเรื่องยากให้ยืนยันก่อนใช้เพชรเยอะ
