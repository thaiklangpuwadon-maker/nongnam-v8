# Ideas เพิ่มเติมให้เหมือนมนุษย์ขึ้น

ยังไม่ควรลงพร้อมกันทั้งหมด ให้ค่อย ๆ เพิ่มทีละชั้น

## 1) Breath / Rhythm Memory

จำว่าเมื่อกี้น้องน้ำพูดค้างไหม เช่น:
- เมื่อกี้หยุดแค่ “อุ้ย”
- เมื่อกี้สวนกลับแล้ว
- เมื่อกี้งอนอยู่
- เมื่อกี้พูดจริงจัง

เพื่อให้ bubble ต่อไปสอดคล้อง ไม่เหมือนสุ่มใหม่หมด

## 2) Conversational Momentum

วัดว่าบทสนทนากำลังไหลไหม:
- ถ้าโต้กันไว → เพิ่มโอกาสพูดแทรก/สวน
- ถ้าผู้ใช้เงียบไปนาน → ลดความรัว
- ถ้าเรื่องจริงจัง → ห้ามรัวเล่นมาก

## 3) Self-Repair

คนจริงพูดผิดแล้วแก้คำเอง:
```txt
ไม่ใช่ ๆ น้ำหมายถึง…
เดี๋ยวนะ เมื่อกี้พูดแรงไปนิด
เอาใหม่ น้ำอธิบายแบบนี้ดีกว่า
```

## 4) Micro Silence

บางครั้งไม่ตอบทันที:
- เงียบเพราะงอน
- เงียบเพราะเขิน
- เงียบเพราะกำลังคิด
- เงียบเพราะไม่อยากยอมแพ้

แต่ต้องไม่ค้างจนดูบั๊ก

## 5) Relationship Tempo

ถ้าสนิทมาก:
- เถียงได้มากขึ้น
- แซวแรงขึ้นนิด
- มีประชดน่ารัก
- พูดสั้นแบบรู้ใจกัน

ถ้ายังไม่สนิท:
- สุภาพกว่า
- ลดการสวน
- ลดความหึง/งอน

## 6) Innuendo Understanding

เข้าใจคำล่อแหลม แต่เลือกตอบ:
- รู้ทันแต่เฉไฉ
- เขินแล้วดุ
- เล่นต่อแบบไม่โจ่งแจ้ง
- ปฏิเสธนุ่ม ๆ ถ้าเกินขอบเขต

## 7) Body-State Coupled Speech

สภาพร่างกายกระทบจังหวะ:
- ง่วง → พูดช้า/สั้น/งอแง
- หิว → หงุดหงิดง่าย/อ้อนของกิน
- สดชื่น → แซวไว
- เหนื่อย → ตอบน้อยแต่มีอารมณ์
